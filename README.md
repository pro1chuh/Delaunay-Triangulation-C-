# Delaunay Triangulation (C++)

Реализация алгоритма триангуляции Делоне на языке C++.

Алгоритм строит триангуляцию для набора точек на плоскости и сохраняет результат в SVG-файл.

## Возможности

- построение триангуляции Делоне
- генерация случайных точек
- экспорт результата в SVG
- визуализация треугольной сетки

## Алгоритм

Используется алгоритм **Bowyer–Watson**.

Основные шаги:

1. Создание супер-треугольника
2. Последовательная вставка точек
3. Удаление "плохих" треугольников
4. Перестроение сетки
5. Удаление треугольников супер-треугольника
